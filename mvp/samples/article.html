A Tiny Browser Engine

by Foysal, written as a weekend MVP.

What it does

It parses a small subset of HTML, applies CSS rules, lays out block and inline boxes, and paints the result to a PNG using tiny-skia for rectangles and fontdue for glyphs.

What it doesn't do

No flexbox, no grid, no scripting, no network. Just a straight render from HTML and CSS to pixels.

This is a note block styled with a colored background and padding.
