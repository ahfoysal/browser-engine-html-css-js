ENGINEERING NOTES

Shaping text, the hard way

by a curious afternoon — April 2026

Real typography needs a shaper. Without one, the gap between AV is identical to the gap between AL, and the word Kerning looks wrong in a way you can feel but cannot quite name. The M3 engine uses rustybuzz, a pure-Rust port of HarfBuzz, to do this work properly.

“A good shaper is invisible. You only notice its absence.”

Each inline run is converted into a stream of glyph ids with precise x-advances. Those advances pick up pair kerning automatically, which you can see in headings set at large sizes. The ids are passed to fontdue for rasterization, and the two libraries agree on the glyph table because they read the same .ttf bytes. NEW

Font families fall back through the usual list: serif resolves to DejaVu Serif, monospace to DejaVu Sans Mono, everything else to DejaVu Sans. Italic runs — like this one — use a real oblique face rather than a synthetic slant. Bold remains a second-pass smear for now; a real bold family is bundled and used automatically for strong elements.
